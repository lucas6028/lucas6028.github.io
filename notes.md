# 名詞解釋

## 翻譯

| 英文        | 中文     | 解釋                                                                                                                                                                                           |
| ----------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| icosahedron | 二十面體 | 是 Three.js 中的一種幾何體，它創建了一個二十面體（正二十面體）的幾何形狀。這種幾何體由 20 個等邊三角形組成，是一種常見的多面體。                                                               |
| decal       | 轉印貼紙 | 在 3D 圖形中，"decal" 是指一種貼花技術，它允許你將一個紋理（通常是一個圖像）應用到一個 3D 模型的表面上。這種技術通常用於在模型上添加細節，如標誌、標籤或其他圖案，而不需要修改模型的幾何形狀。 |
